# 📚 Guías de Usuario - Leon Express

## Bienvenido a las Guías de Usuario de Leon Express

Este directorio contiene guías completas y fáciles de entender para todos los usuarios de la plataforma Leon Express, diseñadas especialmente para personas no técnicas.

---

## 📖 Guías Disponibles

### 1. 👨‍✈️ Guía para Conductores (Drivers)
**Archivo**: `GUIA_USUARIO_DRIVER.md`

Esta guía está diseñada para los conductores que usan la plataforma para:
- Ver sus entregas asignadas
- Consultar sus recolecciones
- Revisar sus rutas
- Ver sus ganancias
- Gestionar su información personal

**¿Cuándo usar esta guía?**
- Si eres conductor y necesitas aprender a usar la plataforma
- Si tienes dudas sobre cómo ver tus entregas o ganancias
- Si necesitas ayuda para iniciar sesión o cambiar tu contraseña

**Contenido incluye:**
- ✅ Cómo iniciar sesión
- ✅ Cómo usar el Dashboard
- ✅ Cómo ver y gestionar entregas
- ✅ Cómo ver recolecciones
- ✅ Cómo consultar rutas
- ✅ Cómo ver ganancias
- ✅ Preguntas frecuentes

---

### 2. 👨‍💼 Guía para Administradores
**Archivo**: `GUIA_USUARIO_ADMIN.md`

Esta guía está diseñada para administradores que gestionan toda la operación:

- Creación y gestión de rutas

- Administración de paquetes y entregas

**¿Cuándo usar esta guía?**
- Si eres administrador y necesitas aprender todas las funcionalidades
- Si necesitas crear usuarios, rutas o gestionar vehículos
- Si tienes que revisar información OCR o gestionar finanzas

**Contenido incluye:**
- ✅ Cómo iniciar sesión como administrador

## 🚀 Cómo Usar Estas Guías

### Para Principiantes
1. **Empieza por el principio**: Lee la sección "Primeros Pasos" de tu guía correspondiente
2. **Sigue el orden**: Las guías están organizadas de manera lógica, desde lo básico hasta lo avanzado
3. **Usa la tabla de contenidos**: Salta directamente a la sección que necesitas usando el índice
4. **No te apresures**: Tómate tu tiempo para entender cada sección antes de continuar

### Para Usuarios con Experiencia
1. **Usa la tabla de contenidos**: Ve directamente a la sección que necesitas
2. **Busca en el documento**: Usa la función de búsqueda (Ctrl+F o Cmd+F) para encontrar términos específicos
3. **Consulta las preguntas frecuentes**: Muchas respuestas comunes están en la sección de FAQ

### Para Capacitadores
1. **Usa las guías como material de capacitación**: Puedes seguir el orden de las guías para capacitar a nuevos usuarios
2. **Personaliza según necesidades**: Adapta las explicaciones según el nivel de conocimiento de tus usuarios
3. **Combina con práctica**: Alterna la lectura de la guía con práctica real en la plataforma

---

## 📱 Formato de las Guías

Las guías están escritas en formato Markdown (.md) y pueden ser leídas:
- **En cualquier editor de texto**: Visual Studio Code, Notepad++, etc.
- **En GitHub**: Si subes los archivos a un repositorio
- **En plataformas de documentación**: GitBook, Notion, etc.
- **Impresas**: Puedes convertirlas a PDF para imprimir

### Convertir a PDF (Opcional)
Si quieres convertir las guías a PDF para compartirlas o imprimirlas:
1. Usa un convertidor online de Markdown a PDF
2. O usa herramientas como Pandoc: `pandoc GUIA_USUARIO_DRIVER.md -o GUIA_USUARIO_DRIVER.pdf`

## 💡 Consejos para Usuarios

### Si es tu primera vez:
1. ✅ Lee la sección "Primeros Pasos" completamente
2. ✅ Practica iniciando sesión varias veces
3. ✅ Explora el Dashboard antes de hacer cambios
4. ✅ No tengas miedo de hacer preguntas

### Si tienes problemas:
1. ✅ Revisa la sección "Preguntas Frecuentes" de tu guía
2. ✅ Busca en el documento usando Ctrl+F (o Cmd+F en Mac)
3. ✅ Contacta a tu supervisor o administrador
4. ✅ Toma capturas de pantalla del problema para explicarlo mejor

### Si encuentras un error en la guía:
1. ✅ Notifica al administrador del sistema
2. ✅ Indica qué sección tiene el error
3. ✅ Sugiere cómo debería decirse correctamente
